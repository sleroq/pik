# Api server for Pik

## Usage

1. Install dependencies
   ```bash
   yarn
   ```
2. Create end modify `env.bash`
   ```bash
   cp scripts/env.bash.example scripts/env.bash
   ```
3. Start the server
   ```bash
   scripts/run.bash
   ```

Query example: `http://localhost:3333/packs/?userId=@sleroq:sleroq.link`
