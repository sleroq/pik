## Usage

```bash
$ yarn
```

```bash
$ cp .env.example .env
```

```bash
$ yarn preview
```

## Development

### Develop

```bash
yarn dev
```

### Lint & format

```bash
yarn lint:fix
```

### Preview

```bash
yarn preview
```

### Build

```bash
yarn build
```

You can deploy the `dist` folder to any static host provider (netlify, surge, now, etc.)
